# Villa Manager Backend - Spring Boot API

Complete REST API backend for Villa Manager Pro application.

## Setup

```bash
cd villa-manager-backend
docker-compose up -d
mvn clean install
mvn spring-boot:run
```

API running at: http://localhost:8080/api
